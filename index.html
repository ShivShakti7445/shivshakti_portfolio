Projects

Presented below are examples of my mobile development projects crafted utilizing Flutter, Android Studio, Figma, and Firebase.

Schedulrr

Schedulrr is a smart scheduling web app that helps users manage their daily plans and events efficiently. Users can create, edit, and delete tasks, organize their schedules visually, and stay on track with time-sensitive activities. The app offers a clean interface with responsive design, built using React.js and Tailwind CSS.

Edemy

This web-based learning platform allows users to explore, enroll in, and complete online courses. It features integrated video playback (via YouTube), user authentication, and course progress tracking. The clean, responsive interface provides a structured learning experience, with categories and filters to help users find relevant content efficiently. Built using React.js, Tailwind CSS, Vite, and Clerk for secure login and user management.

Micromouse Maze Game

This interactive game simulates a Micromouse navigating a maze. Users can start, reset, and watch the mouse find the shortest path using pathfinding algorithms. It offers a visual learning experience for understanding maze-solving logic and algorithmic movement. The game is built for educational and entertainment purposes.

Trip Planning App

This application will offer users features such as Google authentication for sign-in and sign-up processes. Additionally, users can create comprehensive trip itineraries by adding all necessary items. Should they overlook any essential items, they can seamlessly share their checklist with friends, enabling collaborative additions to ensure nothing is overlooked.

ToDo-App

This application enables users to independently create their personalized checklists. Users can create, edit, delete, and restore items within the checklist as needed. Furthermore, they can mark all completed tasks and unmark them if necessary.

Tic Tac Toe App

This application facilitates users to engage in a game of Tic Tac Toe with their friends, enhancing their entertainment experience. Additionally, the application announces the winner of each game, after which players have the option to restart the game for another round of play at their discretion.
